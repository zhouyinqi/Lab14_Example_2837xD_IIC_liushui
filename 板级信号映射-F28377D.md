# F28377D 板级信号映射

数据来源：`系统主控制器DSP&FPGA引脚分配-V01-20260313(2).xls`，版本日期 2026-03-13。本文只记录表内明确给出的信号，不推测未列出的 ADC 通道或 FPGA 寄存器地址。

## 已确认的 DSP1 外设

| 外设 | 软件映射 | 用途 |
| --- | --- | --- |
| I2C-A | SDA GPIO42，SCL GPIO43，INT GPIO114 | PCA85073 实时时钟 |
| I2C-B | SDA GPIO2，SCL GPIO3，INT GPIO6 | TMP116 板卡温度 |
| CAN-B | RX GPIO13，TX GPIO12 | J10 对外 CAN |
| SCI-A | DIR GPIO133，TX GPIO135，RX GPIO137 | 手持设备 RS-485 |
| SCI-B | DIR GPIO11，TX GPIO14，RX GPIO15 | J10 对外 RS-485 |
| SPI-A | GPIO16～19 | 旋变接口 |
| SPI-B | GPIO24～27 | SD 卡接口 |
| SPI-C | GPIO122～125，FRAM CS GPIO33 | W25Q64 Flash、FM25V20A FRAM |
| EMIF1 | 16 位数据、18 位地址，CS2 GPIO34，CS3 GPIO35 | W5300 与 IS61LV25616AL 共用总线 |
| EMIF2 | 16 位数据、12 位地址，CS2 GPIO116 | DSP1 与 FPGA 通信 |
| ePWM1～12 | A 路 GPIO145/147/.../167，B 路 GPIO146/148/.../168 | 24 路信号直连 FPGA |
| LED | GPIO144、GPIO143 | DSP 程序灯 D34、D35 |
| Watchdog | GPIO95 | 外部看门狗输入 |

全部数值宏位于 `include/Board_Pinmap.h`。

GPIO144/D34 和 GPIO143/D35 已接入 `BOARD_TEST_ID_GPIO`。只有收到测试命令时才交替输出，测试后恢复输入状态；不会在上电空闲状态自动闪烁。

## 需要继续确认

1. HPD 大纲中的 DI3 位于 FPGA 侧 `F_DI3`，不是 DSP GPIO；需要 EMIF2 寄存器协议后才能读取。
2. FPGA 表只列出 `F_DO0`、`F_DO1`、`F_DO2`。必须确认大纲所称“DO3”是否对应零起始编号的 `F_DO2`，当前不做等同假设。
3. 母线电压、线电压、相电流和 6 路 NTC 的 ADC/调制器通道没有出现在本引脚表中；需要模拟采样原理图或 FPGA 采样寄存器表。
4. ePWM1～12 只确认到 DSP-FPGA 连接，未给出 FPGA 到两套 HPD 绕组/门极驱动的最终路由，当前禁止发波。
5. W5300 的中断、复位和 BRDY 标记为 `D2_GPIO0/1/94`，属于 DSP2/CPU2；CPU1 固件不能直接按普通 CPU1 GPIO 使用。
6. FPGA 的 DI/DO、STO、光纤和模拟 IO 需要 FPGA 地址映射与位定义，才能纳入一键测试。
